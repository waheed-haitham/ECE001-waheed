Linux Operating System

Unlike many other operating systems, development on Linux isn't led by any one company.
The operating system was created by Finnish programmer Linus Torvalds in 1991.
Nowadays, programmers from all over the world collaborate on its open source code and submit tweaks to the central kernel software and other programs.
A wide assortment of commercial and open source software is available for Linux, and various Linux distributions provide custom user interfaces and tools for installing
software onto machines running the operating system.
A favorite of many programmers, Linux is widely used on corporate and scientific servers, including cloud computing environments.
Linux can be run on a wide variety of hardware and is available free of charge over the internet.
