Microsoft windows

Microsoft Windows has existed in one form or another since 1985, and it remains the most popular operating system for home and office computers.
Its latest versions, including Windows 10, are also used on some tablets, and the OS is used on some web and number-crunching server computers as well.
Computers from a wide variety of manufacturers can use Windows.
Initial versions of Windows worked with an earlier Microsoft operating system called MS-DOS, providing a modern graphical interface on top of DOS's traditional text-based commands.
